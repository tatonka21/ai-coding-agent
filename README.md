# 🤖 AI Coding Agent — Web App Builder

An AI-powered web application builder. Describe what you want, and the AI builds it.

## Features (coming)
- 🖥️ **Monaco Code Editor** (VS Code quality) with syntax highlighting
- 📂 **Multi-file Project System** (HTML, CSS, JS tabs)
- 👁️ **Live Preview** — See your website update in real-time
- 🤖 **AI Build** — Describe an app and AI generates the whole project
- 🛠️ **AI Edit & Refactor** — Select code and tell AI what to change

## Quick Start
```bash
npm install
npm start
# Open http://localhost:3001
```

## Requirements
- [Node.js](https://nodejs.org/) v18+
- [Ollama](https://ollama.ai/) running locally
